# 🍳 SongShip - frontend

Frontend of a music transfert application designed to transfert playlist from deezer to spotify

---

## 🛠️ Tech Stack

- React
- Typescript
- HTML/CSS3
- REST API consumption

---

## ⚙️ Features

-Connexion to streaming plateform profile
-Show all your playlists
-Transfer it to spotify
-Component-based architecture
-API integration with a backend service

---

## ⚙️ Getting Started
Prerequisites

-Node.js
-npm

## Installation
```bash
git clone https://github.com/WDFAllan/songshipfront.git
cd songshipfront
npm install
npm start
```
